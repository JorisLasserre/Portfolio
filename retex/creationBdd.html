Création d'une base de données

Nous avons eu à concevoir une base de données répondant à un besoin client

Information sur le projet

  • Catégorie : Base de données
  • Langages : SQL
  • Logiciel : SQL Server, WinDesign
  • Date : Janvier 2023
  • Taille d'équipe : À deux

Description du projet

Le point de départ était une liste d'exigence d'un client, nous avons dû filtrer les informations utiles afin de créer un schéma entité-association sur winDesign tout en respectant les différentes contraintes (Unicité, contextuelle ...). Nous avons ensuite créé la base de données puis nous y avons ajouté des éléments, tout cela en SQL. Durant ce projet la partie gestion de projet était noté ce qui nous a appris l'utilisation d'un diagramme de Gantt.
