Biosphère7

Le projet Biosphère se composait de 2 phases: la 1ère consistait à coder les règles d'un jeu de plateau tandis que la deuxième consistait à créer une IA pour y jouer.

Information sur le projet

  • Catégorie : Application console
  • Langages : Java
  • Date : Décembre 2022 - Janvier 2023
  • Taille d'équipe : Seul / À deux

Description du projet

Le projet biosphère est un jeu de plateau original où plusieurs actions sont possibles à chaque tour. Le but étant de gagner en aillant plus de points que son adversaire. Lors de ce projet j'ai appris à implémenter des algorithmes d'IA notamment l'algorithme MinMax, à travailler sur des matrices ainsi que sur un projet déjà existant. Lors de la deuxième parti j'ai appris les bases de git pour travailler à deux sur le projet.
